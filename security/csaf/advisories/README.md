# Published Advisories

Place published CSAF advisory JSON documents in this directory.

Suggested naming:

- `CVE-YYYY-NNNNN.json`
- `GHSA-xxxx-xxxx-xxxx.json`
- `containd-YYYY-001.json` when a CVE/GHSA identifier is not yet assigned

Each published document should describe:

- the vulnerability and scope
- affected products / versions
- fixed versions
- mitigations or workarounds
- references to releases, commits, CVEs, GHSAs, and external analysis as appropriate
